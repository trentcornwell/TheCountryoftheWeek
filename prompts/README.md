# Prompt Library

This directory will hold reviewed, versioned prompts for bounded editorial or maintenance tasks such as source extraction, draft summaries, accessibility review, and fact-change reports.

Each prompt must declare its purpose, inputs, required output schema, prohibited assumptions, source/citation requirements, human review step, and version. Prompts must not contain secrets or copied copyrighted source material. AI output is draft material and may not be published without human fact-checking, editorial review, and rights review.
