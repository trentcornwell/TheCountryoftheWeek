# Data

This directory is reserved for reviewed canonical manifests, schemas, aliases, source registries, and small non-sensitive fixtures.

The future `countries.json` must include a schema version, manifest version, effective date, immutable stable keys, CIA identifiers, display names, normalized sort keys, ISO codes where applicable, and source/retrieval metadata. Its first rotated entry must resolve to Kiribati. Before it is created, the project must approve an eligibility policy for dependencies, territories, disputed entities, and CIA naming changes.

Bulk scraped content, licensed media, WordPress exports, and secrets do not belong here.
