# Country Week Theme

This directory contains the custom WordPress theme. The current PHP and CSS files are the pre-existing starter prototype relocated from the repository root during Phase 1; they are not a finished design or scheduler implementation.

The theme will own presentation, template composition, theme supports, and minimal front-end assets. Reusable Country registration, schedule calculation, import behavior, and canonical data should live outside templates and eventually outside the theme in a focused first-party domain plugin.
